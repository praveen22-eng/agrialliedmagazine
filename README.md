# Agri Allied Magazine

A professional peer-reviewed journal publication platform for agricultural publications and food systems research.

## 🚀 Quick Start

### Option 1: Launch Static Website (Fastest)
1. View the website: Open `journal_publication_system.html` in your browser
2. Customize with your journal details
3. Deploy to GitHub Pages or any web host

### Option 2: Full Backend Setup (Production)
See `QUICK_START_GUIDE.md` for 30-minute setup guide.

## 📋 Features

✅ Professional journal website template  
✅ Article submission portal  
✅ Author dashboard  
✅ Peer review workflow  
✅ Full-text search  
✅ Citation export (APA, MLA, Chicago, BibTeX)  
✅ Responsive mobile design  
✅ Admin dashboard  
✅ Database schema (PostgreSQL)  
✅ Node.js/Express backend examples
